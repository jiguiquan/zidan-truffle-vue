# zidan-truffle-vue

#### 介绍

本项目作为truffle + vue项目的纯净整合，以后的项目可基于此项目创建

#### 环境声明：

1. 需要有truffle环境，已安装的可以忽略
2. npm -g install truffle
3. npm install -g ethereumjs-testrpc

#### 软件架构
软件架构说明
truffle 
webpack
vue

#### 使用说明
首先保证在zidan-truffle-vue目录下
1.  truffle devlop  *[·-·]:运行truffle本地私有网络 http://127.0.0.1:9545
2.  compile         *[·-·]:编译
3.  migrate         *[·-·]:部署
使用另一个终端
4.  cd vuebox       *[·-·]:进入vue项目
5.  npm install     *[·-·]:安装依赖
5.  npm run serve   *[·-·]:启动服务
